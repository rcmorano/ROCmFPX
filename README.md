# ROCmFPX Handoff

This repository branch is the review and handoff surface for the experimental
ROCmFPX family.

The detailed operating notes live in [docs/ROCmFPX-HANDOFF.md](docs/ROCmFPX-HANDOFF.md).

This branch is intentionally documentation-first. The implementation work lives
in the local llama.cpp experiment tree that was used to produce the current
ROCmFPX/ROCmFP4 behavior.
